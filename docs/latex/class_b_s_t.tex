\doxysubsection{Detailed Description}
Class representing every \mbox{\hyperlink{class_b_s_t}{BST}} node and the tree (every instance of this class could be a new \mbox{\hyperlink{class_b_s_t}{BST}} root). This class implements basic \mbox{\hyperlink{class_b_s_t}{BST}} operations\+: insertion, deletion and search.
